# BarberApp-API
API de BarberApp

Package que se usaron:
* Restify
* Knex
* Objection
* jsonwebtoken
* dotnet

Debes tener instalado:

* Docker
* Docker composer

Aplicar el siguiente codigo si quieres correrlo.

> docker-compose run app
> docker-compose up

si quieres buildearlo y correr la app..

> docker-compose up -d
